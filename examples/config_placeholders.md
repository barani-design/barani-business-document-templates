# Configuration placeholders

Search the scripts for these placeholders and replace them for your company:

- `XX00 0000 0000 0000 0000 0000` — IBAN with spaces
- `XX0000000000000000000000` — compact IBAN
- `YOURBICXXX` — BIC/SWIFT
- `Your Company s.r.o.` — company name examples in docs only
- report labels if you do not want the BARANI naming style

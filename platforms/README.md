# Platform implementations

This repository is versioned by platform and application version.

Current implementations:

- `odoo/16/` — working Odoo 16 server-action/QWeb implementation.
- `odoo/19/` — placeholder for future Odoo 19 implementation.

The shared process-flow documentation is kept in the root `docs/` folder. Version-specific server actions, extracted XML templates, probes, and restore tools live under their platform/version folder.

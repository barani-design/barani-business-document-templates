# Fiscal positions and taxes

Use fiscal positions to map the included down-payment source tax:

- Domestic: no mapping needed if product has SK included advance tax
- Intra-EU B2B: included advance tax -> 0% EU B2B
- Non-EU export: included advance tax -> 0% non-EU export
- OSS B2C: included advance tax -> included destination-country OSS advance tax

Populate fiscal-position notes; the invoice template prints them.

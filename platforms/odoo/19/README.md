# Odoo 19 templates (planned)

This folder is reserved for the Odoo 19 implementation of the same business-document families.

Target document families:

- VAT invoices / RI / DPI / credit notes
- Quotations / Sales Orders / Pro-forma invoices
- Delivery notes
- Pick lists

Do not copy Odoo 16 server-action installers here without testing against the Odoo 19 QWeb/report API and model fields.
